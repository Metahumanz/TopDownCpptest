###### 2.无法右键点击TopDownCpptest.uproject→**生成Visual Studio项目文件**

如果该文件图标不是UE图标，那么ue还没有和系统关联，再次打开Epic Games Launcher有可能提示实现关联。然而仍然不能解决，暂时使用控制台解决。

但在此之前一定要文本检查编辑TopDownCpptest.uproject，

如果原来是这样：

```json
"Plugins": [
  {
    "Name": "VisualStudioTools",
    "Enabled": true,
    "SupportedTargetPlatforms": [
      "Win64"
    ]
  }
],
```

那么Plugins替换成 **空数组**（删除所有插件配置）

```json
"Plugins": [],
```

因为vs2019和ue4.24不支持VisualStudioTools,直接生成项目文件出问题。



然后控制台生成Visual Studio项目文件

1. 按 `Win+R` 快捷键，输入 `cmd` 并回车，打开命令提示符（CMD）；
2. 在 CMD 中输入命令，先切换到 UE4 的 \UE_4.24\Engine\Binaries\DotNET\

再执行

`UnrealBuildTool.exe -ProjectFiles -Project="你的UE项目路径\XXX.uproject" -Game -Engine`

对于我来说一句就可以,即切换UnrealBuildTool.exe所在目录和运行该exe文件

```cmd
D:\program\UE_4.24\Engine\Binaries\DotNET\UnrealBuildTool.exe -ProjectFiles -Project="D:\UN\TopDownCpptest\TopDownCpptest.uproject" -Game -Engine
```

然后得到.sln文件
